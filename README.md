## NSQ Docs

https://nsq.io/

### Development

Offline documentation requires [jekyll][jekyll]:

```bash
$ bundle install
$ bundle exec jekyll serve --safe --watch --baseurl ''
# site is available at http://127.0.0.1:4000/
```
